# PBLMobile
# TaskManagement
## Features:
1.Account Admin:
- CRUD products
-  Accept/Cancel orders
-  Show product information
-  Search product
-  Sign in/Sign out
2. Account User:
-  Sign in/Sign out/Sign up
-  Show product information
-  Update personal information
-  Reset password
-  Search products
-  Order products
